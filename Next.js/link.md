# Navegar entre path propio
Utilizando next/link evitamos que la pagina recargue ganando eficiencia.

Primero importamos 

```js
import Link from "next/link";
```

Implementacion

```js
        <Link href='/contact'>
          <a>Contact Next</a>
        </Link>    
```
La etiqueta <a> es obligatoria.
